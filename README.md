# ft_linear_regression

ft\_linear\_regression project of school 42

## Usage

```
cd src
python cli.py --help
```

### Train model

```
python cli.py train
```

### Output model cost

```
python cli.py cost
```

### Plot model and dataset

```
python cli.py plot
```
